### 주요 기능

- 주문 등록
- 결제하기
    - 할인 : 수량의 합이 10개가 넘는 경우 10개당 10,000씩 할인
        - 10개는 10,000원 할인, 20개는 20,000원 할인
    - 현금 할인 : 5%할인
- 프로그램 종료

### 상세 기능

- 주문 등록
    - 테이블에 해당 메뉴를 수량만큼 추가한다.
- 결제하기
    - 신용카드 : 수량할인
    - 현금 : 수량할인 & 현금할인

### 플로우

1. 기능을 입력 받는다.
2. 각 기능을 수행한다.

### 유효 검사

- 테이블당 최대 주문 수량은 99개이다.
